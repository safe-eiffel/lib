Cluster ecli.abstract

Abstract Cluster

ECLI core classes.

ECLI_SESSION instances represent a connection to a database

ECLI_STATEMENT is the ancestor of all classes that allow execution of SQL statements.

ECLI_SHARED_ENVIRONMENT give access to connection pooling. Enabling connection pooling should be the first ECLI operation executed.

ECLI_LOGIN_STRATEGY abstract two possibles strategies:

  • simple, with datasource name, username and password;
  • driver, with a driver-specific connection string, that enable fine setting of connection attributes
