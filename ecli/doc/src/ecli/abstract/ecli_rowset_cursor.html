description

Row cursors that physically fetch sets of rows.%N
Rows are physically retrieved row_count at a time, minimizing network traffic.

library

ECLI : Eiffel Call Level Interface (ODBC) Library. Project SAFE.

-- Execute query definition, positioning cursor on first available result row

ensure then

results_exists: (is_executedandthenhas_result_set)implies(results/=Voidandthenresults.count=result_columns_count)
fetched_columns_count_set: (is_executedandthenhas_result_set)implies(fetched_columns_count=result_columns_count.min(results.count))
