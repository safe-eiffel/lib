Cluster ecli.metadata

Metadata Cluster

Classes that represent various metadata and specific cursor classes that gives access to them. Column metadata are descendants of ECLI_NAMED_METADATA or ECLI_DATA_DESCRIPTION. Other are ECLI_DRIVER, ECLI_SQL_TYPE, ECLI_DATASOURCE.

Cursor classes provide a standard method to query the underlying database catalog. ECLI_METADATA_CURSOR actually is the ancestor of classes that give access to column metadata like table columns, primary or foreign key columns. ECLI_DATA_SOURCES_CURSOR, ECLI_DRIVERS_CURSOR, ECLI_SQL_TYPES_CURSOR

Instances of ECLI_DBMS_INFORMATION give access to various attributes about the underlying DBMS/driver capabilities.
