Arrayed Data Cluster

Classes that represent arrays of data.

For high performance applications developers might want to minimize network traffic. When using arrayed data buffers, a single execute passes arrays of parameters or arrays of results in a single operation.

Those classes should be used with arrayed or rowset-aware classes like ECLI_ROWSET_CURSOR or ECLI_ROWSET_MODIFIER. They all inherit from ECLI_ROWSET_CAPABLE.

Instances of ECLI_ROWSET_STATUS give status information about each single data, while ECLI_STATUS gives status about the global operation.
